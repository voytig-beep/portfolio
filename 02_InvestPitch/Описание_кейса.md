# FlowPilot: инвестиционный pitch deck для AI-финтех стартапа

## Короткое описание для сайта

Инвестиционная презентация для вымышленного B2B SaaS-стартапа FlowPilot. Проект показывает формат pitch deck: проблема, решение, рынок, продукт, бизнес-модель, traction, go-to-market и запрос инвестиций.

## Что должно демонстрировать портфолио

- умение собирать структуру инвесторского питча;
- работу с графиками, метриками и финансовой логикой;
- отличие от коммерческого предложения Noverra;
- более аналитичный, венчурный визуальный язык.

## Форматы результата

План для полного кейса:

- HTML-версия;
- PPTX;
- PDF;
- Figma;
- GIF/video preview;
- изображения ключевых слайдов.
