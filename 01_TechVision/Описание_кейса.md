# Noverra: коммерческое предложение для IT-компании

## Короткое описание для сайта

Премиальная коммерческая презентация для вымышленной IT-компании Noverra Digital Systems. В проект вошли название и логотип, визуальная система, 10 слайдов, HTML-демо, PPTX, PDF, Figma-версия и короткое GIF-превью.

## Описание для карточки портфолио

Задача проекта - показать, как может выглядеть современное коммерческое предложение для технологической компании: без перегруза, с сильными фотографиями, крупной типографикой и понятной бизнес-структурой.

В презентации собран полный путь от первого впечатления до заявки: обложка, позиционирование, проблемы клиента, решение, услуги, преимущества, процесс работы, форматы сотрудничества и финальный контактный слайд.

Результат подготовлен в четырех форматах:

- Figma - для просмотра и дальнейших правок;
- PPTX - для отправки клиенту и работы в PowerPoint;
- PDF - для быстрой демонстрации без редактирования;
- HTML - для интерактивного просмотра в браузере;
- GIF - для короткого превью в мессенджерах и карточках портфолио.

## Что сделано

- разработано название Noverra Digital Systems;
- создан премиальный логотип и базовый набор бренд-ассетов;
- собрана структура коммерческого предложения на 10 слайдов;
- подобраны и обработаны фоновые изображения;
- подготовлена HTML-версия;
- собрана PPTX-версия, совпадающая с HTML по визуалу;
- экспортирован PDF;
- создана Figma-версия с 10 фреймами.
- подготовлено 11-секундное GIF-превью.
